upload()

upload(string  $param_name) : boolean

Upload the file specified

The actual file will be renamed to its md5 hash and saved to the system folder A symlink will be created using the original filename in the specified folder Subsequent file with the same md5 hash will be represented using symlink A 0 byte touch file will also be created to help determine if there are any symlinks left Using ClamAV for anti-virus scan

Parameters

string $param_name

The HTML form input field name

Throws

\Hiokwee\Filemanager\RuntimeException

Any errors encountered during upload

Returns

boolean —

Return true if the upload was successful

See also
PHP client library for ClamAV clamd daemon
Open source antivirus engine
